Pivot the platform architecture away from AI-generated followups.

New product direction:

This app is primarily a:
“Gmail draft automation tool for vehicle shipping brokers using spreadsheet variables.”

Remove dependency on AI-generated followups for core workflow.

---

# NEW CORE WORKFLOW

1. User connects Gmail
2. User writes their own email template
3. User inserts dynamic variables
4. User uploads CSV/XLSX
5. System replaces variables with spreadsheet data
6. Gmail drafts get created automatically

---

# REMOVE AI AS CORE DEPENDENCY

AI should become OPTIONAL only.

Remove AI dependency from:

* draft generation
* followup generation
* campaign creation

Keep AI only for optional helpers:

* rewrite email
* improve tone
* generate subject line

Core functionality must work WITHOUT AI APIs.
